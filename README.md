# REMA Portfolio Website

Professional single-page portfolio website for REMA with Maharashtrian culture and glamour theme.

## Files
- `index.html` - main website
- `style.css` - complete responsive styling
- `assets/` - portfolio images

## How to Run
Open `index.html` directly in Chrome, or use VS Code Live Server.

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Open Vercel and import the repository.
3. Keep framework as Other/Static.
4. Deploy.

## Customization
Change contact email in `index.html` inside the Contact section.
